# Movie Watchlist

## Description
Movie Watchlist is a web application that allows users to create and manage a personal list of movies they want to watch. Users can sign up, log in, add movies, edit them, mark them as watched, and filter their watchlist.

## Features
- User authentication (signup/login/logout)
- Add movies to a watchlist
- Edit movie information
- Delete movies
- Mark movies as watched or unwatched
- Filter movies by watched or unwatched
- Cloud database storage with Firebase Firestore

## Technologies Used
- React
- JavaScript
- Firebase Authentication
- Firebase Firestore
- HTML
- CSS
- Vite
- Netlify (deployment)

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `localhost:5173` in your browser

## Known Issues
- Movie editing currently uses simple prompt boxes instead of a full UI form.
