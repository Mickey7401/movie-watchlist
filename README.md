# Movie Watchlist

## Description
Movie Watchlist is a web app that helps users keep track of movies they want to watch. Users will be able to add movies to a list, mark movies as watched, and remove movies from the watchlist.

## Planned Features
- Add a movie title to the watchlist
- View all movies in a list
- Mark a movie as watched
- Remove a movie from the list
- Show watched and unwatched movies differently
